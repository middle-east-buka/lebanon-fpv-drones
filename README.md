# FPV Drone Strike Locations – Lebanon–Israel Front

Interactive heatmap, cluster map, and timeline visualization for FPV drone strike video records on the Lebanon–Israel front.

This project was generated by **MiddleEastBuka** and builds a geospatial visualization layer on top of an open public catalog.

## Live / Local Files

Main visualization file:

- `index.html`

Supporting data files may include:

- validated CSV table
- location-count CSV
- GeoJSON file
- generated map assets

## What This Project Adds

This project adds an independent visualization and geolocation layer, including:

- extraction and normalization of the README catalog rows;
- manually reviewed location/site names;
- manually confirmed or approximate coordinates;
- heatmap mode;
- cluster-map mode;
- timeline playback;
- daily and cumulative filtering;
- video links inside map popups;
- aggregated counts per mapped location.

The geolocation layer is an analytical addition by **MiddleEastBuka**.

## Methodology Notes

Coordinates in this project are not always exact strike coordinates.

They may represent:

- town or village centers;
- named localities;
- border positions;
- approximate areas based on manual review;
- road / hill / military-position approximations;
- unknown Israeli or border sites where exact identification was not possible.

Rows marked as **Unknown**, **Unknown Israel**, or **Unknown Site** should not be treated as precisely geolocated.
